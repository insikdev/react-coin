# Crypto Tracker

- <a href="https://zzicc12.github.io/react-coin/" target="_blank">링크</a>
- Coinpaprika API 사용
- 추가 예정
  - 다크모드
  - KRW USD toggle

## Features

- `/`
  - 시가총액 상위 100위 암호화폐 정보 및 가격 표시
- `/:id`
  - 암호화폐의 ticker 및 시가총액 정보를 표시
- `/:id/chart`
  - 14일치 데이터(시가, 고가, 저가, 종가)를 candlestick 차트로 시각화
- `/:id/price`
  - 14일치 데이터(종가)를 전일대비 퍼센트로 계산하여 표시

## Dependencies
